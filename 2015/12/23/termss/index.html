Mac下让终端也能扶墙

Posted on 2015-12-23 | In 折腾不止 | | Views:

由于我大天朝官方网的翻云覆雨,导致终端下载html5-tidy插件也是毫无办法。想想还是折腾折腾怎么让终端也能用上ss吧！

Mac自带的html tidy不能识别html5的语法，在语法检查的时候老是各种红点报错，虽然不影响最后的运行，但是这不就是没有让SublimeLinter起到应有的作用了么！不行，必须得下html5-tidy。

$ brew install html5-tidy，等，等，等……咦，进度条不动，还各种报错各种警告，各种换源下载！不能忍呀！看来，是时候让我们大终端也笼罩在自由之光下了！

参照Mac 下安装及配置 ProxyChains-NG 实现终端下代理，配置了ProxyChains-NG：

$ brew install proxychains-ng；

$ open /usr/local/etc/proxychains.conf；

在最后一行加上socks5 127.0.0.1 9050；

把ss设为全局代理；

最后输入proxychains4 brew install html5-tidy，我去，秒速下完了。。。

嗯，全剧终！
