2017都要过了一半，你还回顾2016？

Posted on 2017-05-19 | In 唠唠叨叨 | | Views:

迟来的总结。该总结的还是得总结的。本来打算Q1结束之后在Q2开始的时候写的。也就是4月的时候。谁知一连两个大的项目，中间还好几个需求排队。最终导致一拖再拖。

2016年注定是我人生中关键的一年。

完成了终身大事儿

在2016.6.6这么6的日子里和妹纸领了证~

正式签订了终身不平等条约，把自己卖了再买了个媳妇儿(:з」∠)

顺利完成了转职

运气也比较好，面试的第一家公司就直接拿到了offer。

正式开始了菜鸟前端的职业生涯。

也就是红遍大江南北的海马体照相馆~

在这里我遇到了共同成长的小伙伴们~

经受了一次又一次的挑战。虽然我很坑，但是真的很感谢hold住我的伙伴们~

不知道你们嫌弃不嫌弃我，我反正挺嫌弃自己的。抱歉总是拖大家的后腿！

虽然我在新秀赛季表现的不是特别棒，但是我是相信自己在二年级的表现会经受住战火的考验的。

2016年的计划都完成了么？

答案当然是没有呀！

心太大。

最终没有完成所有的目标。完成度大概是50%。

步子迈得太大，扯着蛋儿了。。。

有在反思——逐渐从信息焦虑症中解脱出来。

更加理性的思考，当然，脸皮也更厚了。

后面我会针对这点专门总结一篇《少就是多，后宫三千我也只独宠你一人——如何在信息洪流中找到自己真正需要的姿势》

是的，先立flag。

虽然2016的完成度仅有一半。但是至少完成了最重要的前面两项。

此外外语学习方面后面不知道怎么滴就断了，然后再也没有拾起来了。

可能原因是：

  • 加班加多了，没有精力再去进行多余的思考和学习
  • 应用场景上的基本需求完成了——能读英文文档和在stackoverflow上找到对应的问题，能基本看懂quroa中的学习，然后对自己的要求不高，于是产生了惰性

由于英文词汇量没有达标，日文学习自然没有按进度进行。

如果说学习英语是为了工作。那日语学习纯粹就是兴趣，希望达到的目标是去日本旅游的时候能掌握最基本的一些场景对话。

慕课网上的前端线路基本完成。极客学院的看了几课感觉质量不符合预期，遂放弃。FCC和百度的IFE的练习则是自己懒癌发作没完成。

Node.js的学习放到了17年的目标里——具体在2017规划中说明。P.S:这里一个明显的大Flag。

2016年主攻方向是前端工程化，在一个个项目中逐渐完成了工程化的迭代，完成了技术栈的更新。

对于工程化这块，我觉得可以后面专门写一篇。是个比较大的话题。名字大概会叫《前端架构就是个笑话，我只是个webpack配置员》。

于是无形中又立了个flag。

在日常开发中使用了ES6，Vue全家桶。基本完成了老旧项目前端工程化构建。

之后的工作将会大幅重构为主，彻底将老旧项目改为新技术栈的代码。

这里耍了个流氓，《没有测试也敢叫工程化？》。没错这是又一个flag。

NG2和React完全没看。没有多余的精力了。Vue都没吃透，先打通这段经脉。。一窍通自然能百窍通，现在还只通了半窍——还是一窍不通。

短暂的拍黄片生涯

16年的7月到9月三个月的时间里我还同时学习了PHP——果然PHP是世界上最好的语言。

三个月的后端经历，浅尝辄止，不过对整个web开发流程有了比较全面的认识。

对个人的帮助还是很大的——至少之后在个人的node项目中至少不会360度花式懵B。（嗯，这里有个flag，而且难度还不小）

关于读书

人丑就要多读书，不过我长得比较帅，所以2016年应该就只读了6本书。

比较遗憾的是，读书的时候欠缺思考，所以读多了最后也变成了天书，完全忘记讲啥了。

学而不思则罔，思而不学则殆。谨记

学以致用做到了，但是缺乏思考带来的是低效能的用功。

我明白这点，不过经常赶工期导致只是会用，却不知道为什么。其实完全可以先多花点时间去思考。后面的效率自然就会高了。

明白的有点晚，愚钝的人呀。

健身计划的搁浅

我好想打自己哦。

君子亦器——关于工作流和工具链

在实践过程中有了新的思路——不必事事记录，不必时时记录。

目前我的工作流的有了新的改进，已经放弃了playtask的成就商店的做法了，原因是欲望都实现以后，这个方式带来的驱动力就没有那么强了。

目前的工具链切换成了Xmind -> Promotodo -> Evernote

详细原因立一个flag——《君子亦器——关于工作流和工具链的探索》

不是总结的总结

我很清楚自己的需求，对自己的未来也有明确的规划，也知道自己的不足。

所以，我需要的只是加强自我管理，坚持执行。

我们的价值观是什么？

  • 诚信
  • 正能量
  • 工匠精神
  • 执行力
  • 团队协作

很幸运，在一个氛围良好的团队中，每天都能获得成长。感谢有你们！

坚持原创技术分享，您的支持将鼓励我继续创作！
