一颗赛艇。

使用yarn

  • 首先使用homebrew安装yarn——$ brew install yarn
  • 然后再把yarn的镜像换成淘宝的镜像
    $ yarn config get registry-> https://registry.yarnpkg.com[查看初始镜像]
    $ yarn config set registry 'https://registry.npm.taobao.org' [这样一来就换成淘宝镜像了，速度绝对是要起飞呀！]
    P.S. 只需要把 yarn 改成npm，同理可以修改npm的镜像地址哟
  • $ yarn init 初始化
  • $ yarn add [package]@[version|tag] 添加依赖，@后面可以省略
  • $ yarn upgrade [package]@[version|tag] 更新依赖
  • $ yarn remove [package] 移除包
  • $ yarn or $ yarn install 安装所有依赖

vue-cli

快速搭建开发脚手架
$ yarn add global vue-cli;
$ vue init <template-name> <project-name> =>eg: $ vue init webpack my-project 初始化项目
