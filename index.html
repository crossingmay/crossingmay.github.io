Hexo更新版本之后deploy报错

发表于 2016-04-14   |   分类于 老军医治八哥   |  

试着把Hexo更新到了最新版本。
在执行 hexo deploy 后,出现 error deployer not found:git 的错误。
只要执行$ npm install hexo-deployer-git --save就可以解决了。是少了依赖的原因。
更新Hexo到3.2之后估计还会有一些问题。更新的内容很不错，集成了许多新的控件哦！值得尝试~
